**2. "Walk me through how you optimized performance by 40%"**

**Answer**:
```
"At Amazon, our customer service page had 5-second load times. Here's my approach:

1. **Profiling**: Used React DevTools Profiler + Chrome Lighthouse
   - Identified: Large bundle (3MB), unnecessary re-renders, no code splitting

2. **Code Splitting**: 
   - Implemented React.lazy() for route-based splitting
   - Reduced initial bundle from 3MB ‚Üí 800KB

3. **Lazy Loading**:
   - Images: lazy loading with Intersection Observer
   - Below-fold components: loaded on scroll

4. **Component Optimization**:
   - React.memo() for expensive components
   - useMemo/useCallback to prevent re-renders
   - Virtual scrolling for long lists

5. **Bundle Analysis**:
   - Webpack Bundle Analyzer to find large dependencies
   - Replaced moment.js with date-fns (smaller)

Result: Load time 5s ‚Üí 3s (40% improvement), LCP improved from 4s ‚Üí 2s

Key lesson: Profile first, optimize based on data, measure impact."
```
